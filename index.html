Concatenation of text (strings)

In most programming langage the + operator when used with text will add text to an existing text, for example : 

                        
let basicText = 'abcde';
print(basicText);
// Output will be: 
// abcde

basicText += 'fgh';
print(basicText);
// Output will be: 
// abcdefgh

let otherText = 'WOUAH';

print(otherText+basicText);
// Output will be: 
// WOUAHabcdefgh

For loops

A loop is a way to do the same thing multiple times. The number of times the loop will go is determined by the programmer. A loop in Javascript looks like that: 

                        
for (var counter = 0; counter < 5; counter++) {
    print ("Hello");
}
// This will print "Hello" 5 times in the output

Explanation of the elements inside the for()

  1. "var counter = 0" : First we create the "counter" that will begin at 0
  2. "counter < 5" : This is the condition to STAY inside the loop. So here the loop is over if counter is equals to 5 or above. Warning if your condition is not correctly set the computer will do an infinite loop and crash
  3. "counter++" : This means that after each loop we add 1 to the counter. "++" is a keyword we use in to add 1 to a number, it's shorter than writing counter = counter + 1 but it is the same.
